    作為自然元素的徵象，爬行巨獸————
無論冰火————一直在人類社會中作為傳奇
而廣為人知。但有傳聞說，還有一種巨獸
與它們相似，卻只棲息於最奇異的環境中。
它們，便是電龍。




<image><drawings_lightningdragon.png 0 80 0 0 194 93 1>






與它們掌握冰或火力量的表親不同，電龍
是夜行性動物，出沒於熱帶草原、方山或
是叢林的漆黑夜空中。
電龍居住在由霆霹方塊構成的巢窠或地下
洞穴中。就像冰龍和火龍非常喜愛某些特
定的寶石和貴金屬一樣，電龍也有它們
愛的礦物，那便是紫水晶和銅。

銅礦分佈在主世界的高海拔地區，而紫水晶
礦埋藏在熱帶草原的地下深處。



銅可以像銀一樣製成各種裝備，但相較而言，
這種金屬的性能要弱得多。
